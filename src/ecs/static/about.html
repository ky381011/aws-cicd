About This Website

This is a static website deployed using ECS with volume mounts.

技術スタック

  • Amazon ECS (Elastic Container Service)
  • EC2 Launch Type
  • Nginx Container
  • Volume Mounts (ホストパス)
  • GitHub Repository (ソースコード管理)

デプロイフロー

  1. EC2インスタンス起動時にuser_dataが実行
  2. GitHubリポジトリから静的ファイルをclone
  3. ファイルを /var/www/html にコピー
  4. Nginxコンテナがボリュームマウントでアクセス
